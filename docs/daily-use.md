# Daily use

The dashboard refreshes every 30 minutes and shows two panes per child plus the
current weather.

{{ img("dashboard.png", "Dashboard") }}

## Reading a recommendation

Each child pane shows:

- A **top** (T-shirt, long-sleeve, jacket, etc.).
- A **bottom** (shorts, pants, dress, leggings).
- Optional accessories (rain boots, scarf — when conditions warrant).

Below the icons is a one-line reason ("Cool with a chance of rain") so you
can glance and trust it.

## Evening / pajama mode

After 7 PM local, or after sunset (whichever comes first), the dashboard
switches to a single PJ recommendation for each child. The next morning it
flips back automatically.

## Manual refresh

Tap any pane to force-refresh the weather pull immediately.
